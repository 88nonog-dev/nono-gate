# Start Here

Quick start for Nono-Gate.

## 1. Run the demo

PowerShell:

./demo/RUN_PROOF_DEMO.ps1

## 2. Verify a decision

./demo/VERIFY_ONLY.ps1

## 3. Understand the output

The system produces:

decision.json  
EVIDENCE_BUNDLE.zip  
governance-ledger.ndjson

These artifacts allow deterministic verification of security decisions.
